**OLS推定量の核心的導出（第7章・必須）：**

```
目的関数：S(β) = (y - Xβ)'(y - Xβ)

偏微分：∂S/∂β = -2X'y + 2X'Xβ = 0
正規方程式：X'Xβ = X'y
OLS推定量：β̂ = (X'X)⁻¹X'y

不偏性の証明：
  β̂ = β + (X'X)⁻¹X'ε
  E[β̂|X] = β + (X'X)⁻¹X'E[ε|X] = β （CLM3より）

分散共分散行列：
  Var(β̂|X) = σ²(X'X)⁻¹ （CLM5より）
```

### 第Ⅳ部：検定理論の体系

**目的：** 統計的検定を体系的に理解・使い分ける

**三大検定の統一理論（第A章）：**

```
χ²分布が親：Z~N(0,1), V~χ²(m)のとき
  t(m)   = Z / √(V/m)
  F(m,n) = (V₁/m₁)/(V₂/m₂)
  t(m)²  = F(1,m)  ← 重要な関係

Wald・LR・LM検定の三位一体：
  H₀のもとで全て～χ²(q)
  有限標本：Wald≥LR≥LM（常にこの大小）
  OLS文脈 → F検定
  MLE文脈 → LR検定
  制約付き推定のみの場合 → LM検定
```

| A | t検定 | t～t(n-k) | βⱼ=βⱼ₀ | 係数の有意性 |
| A | F検定 | F～F(q,n-k) | Rβ=r | 複数制約の同時検定 |

**コクラン=オーカット法の核心：**

```
モデル：yₜ = x'ₜβ + εₜ, εₜ = ρεₜ₋₁ + uₜ

変換（ρが既知の場合）：
  yₜ - ρyₜ₋₁ = (xₜ - ρxₜ₋₁)'β + uₜ

→ uₜはiid → OLS適用可能

反復推定：
  Step1：OLS → 残差êₜ
  Step2：ρ̂ = Σêₜêₜ₋₁/Σê²ₜ₋₁
  Step3：変換して再推定 → 収束まで繰り返す

プレイス=ウィンステン変換：第1観測値を√(1-ρ²)倍
→ 情報損失なし（推奨）
```

**IRFの4種類（第35章）：**

```
① 単純IRF（Reduced Form IRF）：
   ψᵢⱼ(h) = [Φₕ]ᵢⱼ
   → 誤差間の相関を無視。記述的分析にのみ使用可

② 直交化IRF（Orthogonalized IRF）：
   チョレスキー分解 Σᵤ = PP'
   Θₕ = ΦₕP
   → 変数の順序付けに依存（経済理論から正当化必要）

③ 構造型IRF（Structural IRF）：
   Byₜ = B₁yₜ₋₁ + ... + εₜ
   識別制約：n(n-1)/2個必要
     短期制約（Cholesky型）
     長期制約（Blanchard-Quah）
     符号制約（Uhlig 2005）
     外部IV（Proxy SVAR）
   → 経済的な因果解釈が可能

④ 累積IRF（CIRF）：
   CIRF(H)ᵢⱼ = Σₕ₌₀ᴴ IRFᵢⱼ(h)
   → 水準変数への長期累積効果（差分系列で特に重要）
```

**ローカルプロジェクション（第37章）：**

```
各ホライゾンhに独立の回帰：
  yᵢ,ₜ₊ₕ = αʰ + βʰsₜ + Γʰ(L)Wₜ₋₁ + uʰₜ₊ₕ

VAR-IRFとの比較：
  VAR-IRF：効率的だがモデル誤特定が累積
  LP-IRF ：非効率だがロバスト・非線形拡張容易

Plagborg-Møller and Wolf（2021）：
  漸近的にProxy SVARとLP-IVは等価

標準誤差：Newey-West（帯域幅≥h+1）
```

### 第Ⅶ部：パネルデータ分析

**目的：** 個体×時間の構造を持つデータの分析

| 章 | タイトル | 核心論点 |

| 28 | Hausman検定とMundlak | FE・RE選択・時間不変変数の扱い |

| 30 | ダイナミックパネル | Nickell Bias→AB差分GMM→BB System GMM |

**固定効果推定量の核心（第27章）：**

```
Within変換：
  ẋᵢₜ = xᵢₜ - x̄ᵢ  （個体内平均を引く）

β̂ᶠᴱ = (Σᵢ Σₜ ẋᵢₜẋ'ᵢₜ)⁻¹ Σᵢ Σₜ ẋᵢₜẏᵢₜ

→ αᵢが消去される（Within変換の核心）
→ 時間不変変数は推定不可（消えるため）
```

**変量効果モデルのθ（第29章）：**

```
θ = 1 - σ_ε/√(Tσ²_α + σ²_ε)

θ=0 → Pooled OLSと同値
θ=1 → FE推定量と同値
0<θ<1 → 両者の中間

Mundlakアプローチ（Hausman検定の代替）：
  αᵢ = x̄'ᵢξ + ηᵢ として
  yᵢₜ = x'ᵢₜβ + x̄'ᵢξ + ηᵢ + εᵢₜ を推定
  ξが有意 → FE必要
  ξが非有意 → RE使用可
  利点：時間不変変数の効果も推定可能
```

**ダイナミックパネルの流れ（第30章）：**

```
問題：yᵢₜ = ρyᵢₜ₋₁ + x'ᵢₜβ + αᵢ + εᵢₜ

Nickell Bias：plim(ρ̂ᶠᴱ - ρ) ≈ -(1+ρ)/(T-1)
→ T小のとき深刻（T=3で約-50%バイアス）

Anderson-Hsiao（1982）：
  一階差分 + yᵢₜ₋₂をIVとして使用 → 一致推定

Arellano-Bond（差分GMM）：
  モーメント条件：E[yᵢₜ₋ₛ · Δεᵢₜ] = 0 (s≥2)
  → ラグ全体を操作変数として使用
  → 2ステップGMM + Windmeijer（2005）補正SE

Blundell-Bond（システムGMM）：
  差分方程式 + 水準方程式を連立
  → ρ→1でも弱操作変数問題を軽減

報告必須：Sargan-Hansen検定・AR(2)検定・
          操作変数数・1st stage F統計量
```

## 6. 途中式の記述ルール（全章共通）

```

## 7. モデル適合度指標の体系

### 線形回帰
| 指標 | 定義 | 特徴 |
|------|------|------|
| R² | 1 - SSE/SST | 変数追加で必ず上昇 |
| Adjusted R² | 1 - (SSE/n-k)/(SST/n-1) | 自由度修正・追加変数が|t|>1なら上昇 |
| AIC | -2lnL + 2k | 予測精度最大化に適する |
| BIC | -2lnL + k×ln(n) | 真のモデル選択に一致性あり |
| RMSE | √(SSE/n) | 被説明変数と同単位 |

### MLE系モデル
| 指標 | 定義 | 目安 |
|------|------|------|
| McFadden R² | 1 - lnL(full)/lnL(null) | 0.2以上：良好・0.4以上：優秀 |
| Nagelkerke R² | Cox-Snell R²を0〜1に正規化 | ロジスティック回帰でよく報告 |
| AUC | ROC曲線下面積 | 0.7〜0.8：許容・0.9以上：優秀 |
| Hosmer-Lemeshow | χ²(8) | 非棄却→キャリブレーション良好 |

### 必須記載事項チェックリスト

```
全回帰表共通：
□ 標準誤差の種類（括弧内の説明）
□ 有意性の星の基準（*** ** *）
□ 観測数
□ 固定効果の有無（Yes/No）

OLS追加：
□ Adjusted R²
□ F統計量（全体有意性）

IV・2SLS追加：
□ 操作変数名と出所
□ 1st stage F統計量
□ Sargan-Hansen検定（過剰識別の場合）
□ Wu-Hausman内生性検定

ロジット・プロビット追加：
□ 係数か限界効果（AME/MEM）かの明示
□ McFadden Pseudo R²
□ 対数尤度・LR統計量

動的パネルGMM追加：
□ 1ステップか2ステップか
□ Windmeijer補正の有無
□ Sargan-Hansen検定
□ AR(1)・AR(2)検定
□ 操作変数の数
```

### よくある誤りと修正

```
誤り1：ロジット係数を効果量と誤解
  誤：「係数0.5 → xが1単位増でyが0.5増加」
  正：「平均限界効果は0.12（se=0.03）→P(y=1)が12%ポイント増加」

誤り2：「非有意 → 無関係」の誤解
  誤：「n.s.のためxはyに影響しない」
  正：「係数0.02（se=0.04）で非有意だが
       95%信頼区間[-0.06, 0.10]であり
       経済的に有意な効果を排除できない」

誤り3：標準誤差の種類の無記載
  → 必ず「括弧内は○○標準誤差」を注に記載

誤り4：時系列でR²が高い → 良いモデルと判断
  → DW統計量を確認（低ければ見せかけの回帰の可能性）
```

## 9. Stata・R・Python 実装対応表（主要コマンド）

```
